🚀 2024 Vibe Coding 期末專案：旅行公司網頁

🌟 1. 專題願景 (Vibe)

目標對象： 去日本遊玩的旅客。

設計風格： 新日式極簡主義。

核心功能： * 提供五種精選日本 7 天 6 夜旅遊方案。

完整的每日導覽路徑 (Roadmap) 與飯店官方跳轉連結。

推薦自由活動時間的特色餐廳（含必點菜單、聯絡資料）與熱門景點。

支援一鍵導出 PDF 行程表，方便旅客手機隨身查看。

🤖 2. Gemini 協作紀錄 (Prompt Log)

關鍵指令： 你現在是一位專業的前端工程師與 UI 設計師，我想做一個日本 7 天 6 夜的行程，我希望你幫我做成一個網頁，我現在是一位旅遊公司的人想請你幫我們架設網頁。

開發挑戰與解決方案：

內容消失問題： 在優化過程中，部分飯店連結與餐廳詳細資訊曾遺失。透過具體列出清單（飯店網址、推薦菜單、聯絡電話、伴手禮）成功引導 Gemini 恢復全量數據。

渲染報錯修復： 解決了 React Error #130（物件渲染錯誤）以及 SafeImage 未定義等作用域問題。

PDF 導出空白： 修改了隱藏容器的渲染邏輯，確保 PDF 引擎能正確擷取離螢幕外的 HTML 內容。

📂 3. 檔案結構說明

index.html：網頁主要結構與 React 互動邏輯。

style.css：視覺設計、排版美化與動畫效果。

README.md：專案說明文件。

北海道.jpg：用於北海道主題行程的主視覺圖片。

📈 4. 自我心得與反思

Vibe Coding 心得： 透過對話式開發，能快速產出高品質的 UI 佈局，但指令必須精確且持續追蹤數據完整性，否則 AI 容易在執行新任務時「遺忘」舊有的細節。

優化展望： 未來希望能改善大型圖片的載入效率，並整合 Google Maps API 提供即時的店鋪地圖呈現。
